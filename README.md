This repo is an example of what an application handling timers for the kitchen of hope's corner ovens could be
It should used to track and remember the contents of each of the ovens and at what time the hearting starts

It relies on PyQt6
It mainly uses these PyQt tools: QVBoxlayout, QHBoxLayout, QWidget and QTimer
